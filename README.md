# sae202

Ici tous les fichiers mettre sur votre VPS dans le /var/www, attention agence.conf est mettre dans /etc/apache2/conf-available puis mettre les deux commandes suivantes a2enconf agence.conf et service apache2 reload, et le fichier sae202.conf a mettre dans /etc/apache2/sites-available puis entrer les commandes a2ensite sae202.conf pui service apache2 reload. Remplacer dans ces fichiers par votre propre MMI.
